# FedRK

This repository contains code for the experiments in "A federated Kaczmarz algorithm".

main_1a.m: code for the comparison of FedRK, FedAvg, and FedProx (Section 3.1) 

main_1b.m: code for the system with large, unbalanced, and heterogeneous data (Section 3.1)

main_2.m: code for solving the sparse approximation problem (Section 4.1)

main_3.m: code for solving an inconsistent system (Section 4.2)

main_4.m: code for application to the prostate cancer data (Section 4.3)
